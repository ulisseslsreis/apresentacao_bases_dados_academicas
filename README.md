# Bases Acadêmicas de Pesquisa: panorama e uso prático para a Área do Direito

Apresentação desenvolvida pelo **Prof. Dr. Ulisses Reis** (PPGD/UFERSA) com o objetivo de introduzir pesquisadores de graduação e pós-graduação ao uso qualificado das principais bases de dados científicas disponíveis para a pesquisa jurídica.

O material está disponível publicamente para uso por estudantes e pesquisadores de qualquer instituição, como contribuição à democratização do acesso ao conhecimento sobre metodologia de pesquisa acadêmica.

## Acesso aos slides

[Clique aqui para acessar a apresentação](https://ulisseslsreis.github.io/apresentacao_bases_dados_academicas/)

## Conteúdo

- Distinção entre bases de dados, repositórios e indexadores
- Portal de Teses e Dissertações da CAPES
- SciELO, Scopus, Web of Science e Redalyc
- Google Acadêmico: usos e limitações
- Estratégias de busca por tipo de pesquisa
- Pacotes R para coleta automatizada de dados bibliográficos

## Atualizações

O conteúdo é periodicamente revisado e atualizado para refletir mudanças nas plataformas e incorporar novos recursos disponíveis para a pesquisa jurídica.
